# Codebrainhub

A modern platform for sharing app ideas with community feedback.

## Features

- 💡 Share Ideas - Post your app concepts with descriptions and screenshots
- - 🗳️ Voting - Community votes on usefulness (Would Use / Not For Me)
  - - 💬 Comments - Discussion and suggestion flags for feedback
    - - 🔐 Secure - Password authentication with SHA-256 hashing
      - - 🛡️ Hardened - Input validation, rate limiting, XSS protection
       
        - ## Tech Stack
       
        - - Vanilla HTML/CSS/JS
          - - Vite for building
            - - localStorage for data persistence
              - - Web Crypto API for password hashing
               
                - ## Getting Started
               
                - ```bash
                  npm install
                  npm run dev
                  npm run build
                  ```
